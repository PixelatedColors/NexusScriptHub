A lightweight Roblox script hub with a simple UI and fast execution.
## Installation
Install any script executor (ideally: Solara, Volcano, Xeno etc. Use WhatExpsAre.online to find safe executors) get the script from the Github and paste it in your script executor. After pasting, make sure to inject your executor to the Roblox Process. After injection, execute the script and wait 5 - 10 seconds for it to load. After loading, you can use Flux for how long as you want (there wont be a key for a few months)
